# Music Theory App

An interactive music theory and ear training platform. Learn music fundamentals, practice ear training, and master music theory concepts.

## Tech Stack

Framework: Next.js 16 - React 19 with App Router
Language: Typescript
Styling: Tailwind CSS 4 + shadcn/ui

## Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run prepare
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
